Bash Scripting Suite for System Maintenance
Objective
To automate key Linux system maintenance tasks such as backups, updates, and log monitoring using Bash scripting.

Features
Automated system backup (backup.sh)
System update and cleanup (update_clean.sh)
Log monitoring with error alerts (log_monitor.sh)
Main menu interface (menu.sh)
Simple menu-based interface
Logs all actions for auditing
Easy to customize and extend
Usage
Run the main script:

chmod +x menu.sh
./menu.sh
